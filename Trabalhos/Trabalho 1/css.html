Margin

As propriedades para as margens, definem a dimensão de cada uma das quatro margens de um elemento HTML

Propriedades Descrição
margin maneira abreviada para definir todas as 4 margens
margin-bottom define a margem inferior
margin-left define a margem esquerda
margin-right define a margem direita
margin-top define a margem superior

Padding

A propriedade padding se destina a criar um espaçamento interno em um um box entre seus 4 lados e a área de conteúdo do box.

Propriedades Descrição
padding maneira abreviada para definir todos os 4 espaçamentos.
padding-bottom define o espaçamento inferior
padding-left define o espaçamento à esquerda
padding-right define o espaçamento à direita;
padding-top define o espaçamento superior

Valores válidos para as propriedades margin e padding

Valores Descrição
auto O navegador calcula a margem ou espaçamento
lenght define a a margem ou espaçamento em px, pt, cm, etc.
% expressa a margem ou espaçamento em porcentagem.
inherit Especifica que a margem ou espaçamento deve ser herdada do elemento pai

Backgrounds

As propriedades de plano de fundo do CSS são usadas para definir os efeitos de segundo plano para elementos.

Propriedades Descrição
background-color define a cor do fundo
background-image define uma imagem de fundo
background-repeat define a maneira como a imagem de fundo é posicionada
background-attachment define se a imagem de fundo "rola" ou não com a tela
background-position define como e onde a imagem de fundo é posicionada
background-clip define a área do box onde a imagem de fundo é aplicada
background-origin define a posição de origem da imagem no box
background-size define as dimensões da imagem no box
background maneira abreviada para declarar todas as propriedades anteriores

Display

A propriedade display permite que você defina a maneira como determinado elemento HTML deve ser renderizado.

Propriedades Descrição
block faz com que o elemento HTML seja renderizado como bloco ,tal como os parágrafos e os cabeçalhos
inline faz com que o elemento HTML seja renderizado inline, dentro do bloco na mesma linha.
none faz com que o elemento HTML não seja renderizado assim sendo você não o verá
inline-block Um bloco inline é colocado inline (ou seja, na mesma linha do conteúdo adjacente), mas comporta-se como se fosse um bloco.
list-item faz com que o elemento HTML seja renderizado como um item de lista, isto significa que aparecerá um marcador (bullet) na frente de cada item
table faz com que o elemento HTML seja renderizado como tabela.

Alguns valores válidos para as propriedades da lista

Position

A propriedade position especifica o tipo de método de posicionamento usado para um elemento (relativo, absoluto, fixo)

Propriedades Descrição
relative posiciona o elemento em relação a si mesmo. Ou seja, o ponto zero será o canto superior esquerdo, e ele começará a contar a partir dali.
absolute Enquanto o elemento com Position Relative utiliza seu próprio canto para referenciar sua posição, o elemento com Position Absolute se utiliza do ponto superior esquerdo de outros elementos.
fixed irá fixar a posição do elemento na coordenada que você definir. A medida que a página é rolada, o elemento continua fixo na posição que você definiu e o conteúdo da página rola normalmente.

Float

A propriedade float CSS especifica como um elemento deve flutuar.

Propriedades Descrição
right especifica que um elemento deve flutuar para a direita
left especifica que um elemento deve flutuar para a esquerda
